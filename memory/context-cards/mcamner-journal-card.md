# Context Card: mcamner-journal

## Role

Static journal and public writing system for MQ-adjacent thinking.

## Owns

* public journal pages and generated static site
* writing style, voice, and knowledge notes
* terminal-inspired editorial presentation
* public posts about MQ systems and tools
* journal-specific agent read-order rules

## Does not own

* MQ runtime orchestration
* MCP tool execution
* repo health scoring
* durable MQ stack truth
* source-repo implementation details

## Reads from

* local journal markdown and knowledge notes
* generated static site files
* MQ context when writing about MQ systems
* agent entrypoint instructions

## Writes to

* static site pages
* journal posts and knowledge notes
* generated public docs
* no MQ runtime state

## Use this card when

* task involves journal writing or public presentation
* task involves journal knowledge or voice
* task touches generated static site output
* task needs to separate writing from MQ runtime work

## Avoid reading unless needed

* binary image assets
* full generated HTML archives
* unrelated MQ source code
* raw draft notes
